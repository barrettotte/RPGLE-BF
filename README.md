# RPG400-BF
Showing the evolution of RPG by making a BF interpreter in RPG/400, RPGLE Fixed, and RPGLE free.


## Setup
* ensure CCSID correct - ```chgjob ccsid(37)```


## Commands
* build - ```./build.sh```
* Git push - ```git -c http.sslVerify=false push origin master```
